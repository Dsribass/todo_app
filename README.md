## Como usar

Verifique se você possui instalado tanto o [Flutter](https://flutter.dev/docs/get-started/install), como o [Android Studio](https://developer.android.com/studio).


```bash

# Instale as dependências
$ flutter pub get

# Rode o App
$ flutter run

```
